Map based password

It's a new knowledgebased authentication method, where the user's secret is a selected path on a map.

Demo:

It's not a real registration, no server involved, but store some data (username, password hash and salts) in your browser. You accept that, if you "register" in the demo. The data only used and stored on the client-side. For deleting that data click here.

Your browser does not support service workers. This demo not working without it. Sorry!
